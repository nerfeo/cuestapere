La empresa

Una empresa familiar,
una historia real

Cuesta Peré es una empresa familiar dedicada a la producción y venta de miel artesanal. Sus colmenas, ubicadas en entornos naturales, producen dos variedades principales: una miel clara de flores silvestres y una miel oscura de monte bajo y tomillo.

Hasta ahora, la gestión de ventas se realizaba de forma manual — llamadas, mensajes, apuntes en papel. El reto era claro: modernizar el proceso sin perder la esencia artesanal que define a la marca.

Este proyecto nace de esa necesidad real: crear una herramienta que permita vender online, controlar el stock y gestionar pedidos desde cualquier dispositivo, sin complicaciones técnicas.

"La tecnología al servicio de lo artesanal."

El sistema

Cómo funciona

La aplicación cubre el ciclo completo de venta: desde que un cliente descubre el producto hasta que el pedido llega a su puerta.

01 / Cliente
👤
El cliente pide

El cliente pide un bote por tienda online, WhatsApp, llamada o mensaje directo. Sin complicaciones.

02 / Compra
🛒
Reserva de producto

Se registran los datos del pedido. El sistema guarda todo y actualiza el stock.

03 / Preparación
🍯
Preparación del pedido

Se prepara el pedido y queda listo para entregar en el mismo día.

04 / Entrega
🚚
Entrega del pedido

Se entrega en domicilio o punto acordado. Pago en mano, sin comisiones ni intermediarios.

05 / Cierre
✅
Pedido completado

El pedido se marca como finalizado y el stock queda actualizado automáticamente.

Stack técnico

Tecnologías utilizadas

Seleccionadas por ser gratuitas, seguras y fáciles de mantener.

🐍
Flask
Framework web en Python. Gestiona rutas, autenticación, lógica del sistema y comunicación con la base de datos.
🗄️
SQLAlchemy
ORM que permite trabajar con la base de datos usando código en lugar de SQL. Facilita modelos, consultas y migraciones.
💾
SQLite
Base de datos ligera y sin servidor. Ideal para pequeñas empresas: rápida, estable y sin coste de mantenimiento.
🌐
HTML · CSS · JS
Interfaz web simple y rápida. Sin frameworks pesados, máxima compatibilidad y carga instantánea.
🎨
Bootstrap 5
Sistema de diseño responsive. El panel funciona perfectamente en móvil, tablet y ordenador.
📄
xhtml2pdf
Generación automática de facturas en PDF a partir de plantillas HTML del sistema.
🔐
Flask‑Login
Sistema de autenticación seguro. Control de sesiones, permisos y acceso a zonas privadas.
📱
PWA + Service Worker
Modo offline, icono instalable y carga rápida como si fuera una app nativa.
